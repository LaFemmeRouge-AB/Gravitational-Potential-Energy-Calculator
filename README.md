# Gravitational-Potential-Energy-Calculator
Basically what the name says.

User types in the mass, in grams, the acceleration, in m/s^2 (with base of 9.8m/s^2 for base gravity), and the height, in meters. They can either click calculate or just type in the inputs and it will update as they go. It will then say the gravitational potential energy, in joules of course.
